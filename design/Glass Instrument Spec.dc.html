Claude usage widget — Glass Instrument

One design, two themes. Dark Glass and Light Glass swap automatically with the OS appearance (300ms crossfade; time-based fallback 07:00–19:00 light). Default collapsed state shows one ring — the worst enabled meter. Panels in §2 are fully interactive. Read with the README in the handoff bundle; every number here is final.

Window behavior · motion · auto theme
Always-on-top above fullscreen apps, all Spaces/desktops. Never steals focus. No dock/taskbar/Alt-Tab presence; tray icon for quit + re-run setup.

5a · Drag → corner snap
Drag anywhere on pill (or panel header). While dragging: 4 ghost targets fade in (120ms, dashed 1.5px @25%, pill-shaped). Release: snap to nearest corner, 180ms ease-out, 16px margins both axes. Nearest = euclidean distance to corner. Pinned corner persists across restarts; panel always grows inward from it (transform-origin = that corner).

Setting-gated. After 10s without pointer interaction while collapsed: pill slides into the pinned edge leaving a 30×4px bar per visible ring (gap 4, same colors, slide 200ms). Hovering the sliver slides the pill back out. Expanded panel never auto-hides.
